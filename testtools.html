class hydpy.core.testtools.IntegrationTest(element, seqs=None, inits=None)[source]¶

Bases: hydpy.core.testtools.Test

Defines model integration doctests.

The functionality of Test is easiest to understand by inspecting doctests like the ones of modules llake_v1 or arma_v1.

Note that all condition sequences (state and logging sequences) are initialized in accordance with the values are given in the inits values. The values of the simulation sequences of outlet and sender nodes are always set to zero before each test run. All other parameter and sequence values can be changed between different test runs.
